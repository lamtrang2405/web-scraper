Run the crawler UI locally

GitHub Pages can host static files only. The crawler API runs from Flask on your machine.

  • Create virtual environment: python3 -m venv .venv
  • Install dependencies: source .venv/bin/activate && pip install -r requirements.txt
  • Start UI/API: python app.py
  • Open local UI: http://127.0.0.1:5000
